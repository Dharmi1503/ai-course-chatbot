Full Stack Development means building both the frontend and backend of a web application.

Frontend is the part of the website users see and interact with. It includes HTML, CSS, and JavaScript.

Backend is the server side of a web application. It handles data, logic, and database operations.

HTML stands for HyperText Markup Language. It is used to structure content on the web.

CSS stands for Cascading Style Sheets. It is used to style and design web pages.

JavaScript is a programming language that makes web pages interactive and dynamic.

React is a JavaScript library for building user interfaces. It uses components to build web pages.

Node.js is a runtime environment that allows JavaScript to run on the server side.

Express.js is a web framework for Node.js. It makes building backend APIs easier.

A Database is where application data is stored. Common databases are MySQL and MongoDB.

REST API is a way for frontend and backend to communicate using HTTP requests.

Git is a version control system that tracks changes in code and helps teams collaborate.
